enr

本来想写个能够在网站的前端和后台控制台通用的组件，后来就一点点开始写了一些额外的辅助 npm 包。

当时也是为了想着能辅助该包的开发，但是在查资料时发现点开 edge 浏览器新标签页总会有一些讨人厌（其实，是自己耐力不住罢了）的资讯推荐。就想着反正也是练手，就写了两个 edge 插件。然后，发现自己越走越偏。。。。

安装​

使用你喜欢的方式安装该 UI 库。

  • npm
  • yarn
  • pnpm
npm install  enr --save
yarn add enr 
pnpm add enr --save

简介​

该库当前就三两个组件

📄️ 布局

📄️ 自定义钩子

本 ui 库是一个萌芽中的包，就像襁褓中的婴幼儿，既丑又容易早夭。

该库自然跟大佬级别的 antd 是没法比的，甚至比比写的包都要垃圾的多得多。

为了兼容 NextJs ，所以使用了 styled-components 依赖。元宝告我说，就我这个包，等啥时候周下载量在千以上再考虑写独立的网站文档，周下载量再万级别再想着自己写一个 css in js 的包。后来想想，写一个能周下载能到百万的的 css in js 有点不太可能，但是写独立网站文档。。。

还等包下载量到一千。。。。。那不就等不到了
